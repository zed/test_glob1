`test_glob1` - test `glob.glob1()` availability on different Python versions.

USAGE: $ tox

To install `tox <http://codespeak.net/tox/>`_, type: ``$ pip install tox``

python2.4, python2.5, python2.6, python2.7, python3.0, python3.1,
python3.2, pypy, jython executables should be in the PATH.

See ``tox.ini`` for the list of tested Python environment.
